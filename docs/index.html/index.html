▸ one(s, context)

获取单个DOM元素

one(s: (string | object), context: object): (object | null)
Parameters
s ((string | object)) 选择器或DOM元素
context (object) 上下文对象,默认为document
Returns
(object | null): 返回找到的第一个DOM元素,未找到返回null
Example
$.one('#id');  // 返回id对应的DOM元素
$.one('.class', parentEl);  // 在父元素中查找class对应的第一个DOM元素

▸ on(eventType, selector, handler?)

为元素绑定事件监听器,支持事件委托

on(eventType: string, selector: (string | Function), handler: Function?): Object
Parameters
eventType (string) 事件类型,多个事件用空格分隔(如 'click keyup')
selector ((string | Function)) 选择器(用于事件委托)或事件处理函数
handler (Function?) 事件处理函数(当使用事件委托时必传)
Returns
Object: 返回当前jQuery对象实例
Example
// 直接绑定事件
$('.button').on('click', function(e) {
    console.log('clicked');
});

// 事件委托
$('.container').on('click', '.button', function(e) {
    console.log('delegated click');
});

// 绑定多个事件
$('.input').on('keyup keydown', function(e) {
    console.log('key event');
});
▸ off(eventType, selector?, handler?)

移除元素上绑定的事件监听器

off(eventType: string, selector: (string | Function)?, handler: Function?): Object
Parameters
eventType (string) 事件类型,多个事件用空格分隔
selector ((string | Function)?) 选择器(用于移除委托事件)或事件处理函数
handler (Function?) 要移除的事件处理函数
Returns
Object: 返回当前jQuery对象实例
Example
// 移除直接绑定的事件
$('.button').off('click', handler);

// 移除委托事件
$('.container').off('click', '.button', handler);

// 移除多个事件
$('.input').off('keyup keydown', handler);
